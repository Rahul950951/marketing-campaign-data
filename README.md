# Marketing Campaign Data Cleaning

A simple Python project to clean and prepare marketing campaign data for analysis.

## Features
- Standardize column names
- Clean and format numerical fields (e.g., Spend)
- Fix categorical typos (e.g., Channel)
- Normalize boolean values (Active column)
- Convert dates to proper format
- Run integrity checks (Clicks vs Impressions, Start vs End Date)
- Handle outliers in Spend
- Extract features from Campaign Name
